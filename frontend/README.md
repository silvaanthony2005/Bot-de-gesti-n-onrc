# Frontend - Bot CNE

Este directorio contiene el código fuente de la interfaz de usuario.
La arquitectura y estructura de carpetas están documentadas en `../docs/ARQUITECTURA_FRONTEND.md`.

## Estructura Rápida

- `src/components`: Componentes UI reutilizables.
- `src/features`: Módulos de negocio (Chat, Dashboard).
- `src/layouts`: Plantillas de diseño.
- `src/pages`: Vistas principales.
